# Resumen
Este proyecto consiste en una aplicación de escritorio básica desarrollada en Java utilizando JavaFX. El objetivo central de este ejercicio es aprender a vincular componentes de entrada y salida, extrayendo el contenido escrito por el usuario en un campo de texto (TextField) para volcarlo directamente sobre una etiqueta informativa (Label) mediante un evento de botón.

## Características
Sincronización de Componentes: Conecta la entrada de datos con la salida visual de la interfaz de usuario de manera síncrona al pulsar un botón.

Manejo de Propiedades de Texto: Aplica los dos métodos fundamentales de manipulación de cadenas en el entorno gráfico de Java: .getText() para la lectura y .setText() para la escritura.

Layout Secuencial: Agrupa y distribuye los nodos de forma lineal de arriba hacia abajo mediante el contenedor vertical VBox.
